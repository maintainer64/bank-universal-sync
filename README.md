# Bank Sync

Универсальное расширение для браузера Chrome, которое выгружает счета, транзакции и заказы из банковских приложений и
маркетплейсов в CSV, **Sure** или **Firefly III**.

## Поддерживаемые сервисы

- ✅ **Т-Банк**
- ✅ **Яндекс Банк**
- ✅ **Яндекс Лавка**
- ✅ **Жизнь Март**

## Функциональность

- 🔄 **Автоматическая синхронизация** счетов и транзакций в Sure и Firefly III
- 📁 **Выгрузка в CSV** в универсальном формате или формате провайдера
- 🔒 **Безопасное хранение** настроек и данных
- 🎯 **Онбординг** — пошаговая настройка при первом запуске

## Установка

### Способ 1: Автоматическая установка (рекомендуется)

1. Скачайте и запустите скрипт установки:

```bash
curl -s https://raw.githubusercontent.com/maintainer64/bank-universal-sync/main/install.sh | bash
```

Или вручную:

```bash
# Скачайте скрипт из репозитория
curl -fsSL https://raw.githubusercontent.com/maintainer64/bank-universal-sync/main/install.sh -o install.sh

# Дайте права на выполнение
chmod +x install.sh

# Запустите установку (скачает последний релиз с GitHub и распакует)
./install.sh
```

### Способ 2: Ручная установка

1. **Скачайте последний релиз**:

```bash
# Используйте скрипт для скачивания
./install.sh
```

2. **Распакуйте расширение**:
    - Архив автоматически распакуется в `~/bank-universal-sync_latest/`

3. **Добавьте расширение в Chrome**:
    - Откройте Chrome и перейдите в `chrome://extensions/`
    - Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
    - Нажмите **"Загрузить распакованное расширение"**
    - Выберите папку с распакованным расширением

## Настройка

1. Нажмите на иконку расширения в Chrome
2. Перейдите в настройки
3. Настройте:
   - **Тип экспорта** — CSV, Sure или Firefly III
   - **Параметры провайдера** — адрес и токен (если требуется)
   - **Лимит операций** — максимальное количество выгружаемых записей

## Использование

### Синхронизация с провайдером

1. Откройте нужный банковский сервис в браузере
2. Нажмите на иконку расширения
3. Нажмите **"Синхронизировать"** — данные будут отправлены в выбранный сервис

### Экспорт в CSV

1. Откройте нужный банковский сервис
2. Нажмите на иконку расширения
3. Выберите:
   - **"Операции в CSV"** — экспорт транзакций
   - **"Счета в CSV"** — экспорт счетов
   - **"Заказы в CSV"** — экспорт заказов (для маркетплейсов)

## Разработка

### Требования

- Node.js 20+
- Yarn 1.x

### Установка для разработки

```bash
# Клонируйте репозиторий
git clone https://github.com/maintainer64/bank-universal-sync.git
cd bank-universal-sync

# Установите зависимости
pnpm install

# Запустите в режиме разработки
pnpm dev

# Соберите для production
pnpm build
```

### Структура проекта

```
src/
├── index.tsx          # Основной компонент
├── background/        # Фоновые скрипты
├── content/          # Контент-скрипты для банковских сервисов
├── utils/            # Утилиты и хелперы
└── types/            # TypeScript типы
```

## Безопасность

- 🔐 **Локальное хранение** — все настройки хранятся локально в браузере
- 🌐 **Безопасные запросы** — использование HTTPS для API вызовов

## Troubleshooting

### Расширение не работает

1. Убедитесь, что расширение включено в `chrome://extensions/`
2. Проверьте настройки провайдера (адрес, токен)
3. Откройте консоль разработчика (F12) для просмотра ошибок

### Данные не синхронизируются

1. Проверьте подключение к интернету
2. Убедитесь, что выбранный сервис доступен
3. Проверьте правильность настроек в разделе конфигурации
